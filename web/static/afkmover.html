AFK Mover

General

Plugin activation is managed centrally on the plugins page. This page is for configuration only.

-

Auto return after activity

If enabled, users moved by AFK Mover are returned to their previous channel when they become active again.

Excluded channels

Users in these channels are not moved into the AFK channel.

Zielchannel wählen

Wähle den Channel, in den inaktive User verschoben werden.

Excluded channel auswählen

Wähle einen Channel aus, der vom AFK-Move ausgeschlossen werden soll.
